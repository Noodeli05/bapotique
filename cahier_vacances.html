Fiche de cours

La linguistique est la science descriptive du langage humain : elle observe comment les gens parlent réellement, sans prescrire comment ils devraient parler.

Au cœur de la linguistique, le signe réunit un signifiant (forme sonore ou écrite : /ʃa/) et un signifié (concept mental : l'idée du chat). Ce lien est arbitraire — rien dans /ʃa/ ne ressemble à un chat. Preuve : cat, Katze, gato désignent le même animal.

André Martinet a décrit la double articulation : 1ère articulation en monèmes/morphèmes (unités avec sens), 2ème articulation en phonèmes (unités sans sens). Avec ~36 phonèmes, le français génère des millions de mots.

Les niveaux d'analyse : phonologie → morphologie → syntaxe → sémantique → pragmatique. Chaque niveau s'appuie sur le niveau inférieur.

Définitions clés

Signifiant = forme sonore ou graphique du signe  |  Signifié = concept mental associé  |  Référent = objet réel dans le monde

Langue = système abstrait collectif  |  Parole = usage individuel concret

Révision complète — D0 — Introduction & méthode
1. La linguistique : objet et méthode

La linguistique est la science qui étudie le langage humain. Elle ne s'intéresse pas à "bien parler" (c'est le rôle de la grammaire normative) mais à décrire le fonctionnement réel des langues. À retenir Langue = système abstrait partagé par une communauté (le français "en général"). Parole / énonciation = mise en action de la langue par un locuteur dans une situation concrète ("je te parle maintenant"). Analogy : la langue est comme les règles du foot ; la parole est le match joué.

2. Le signe linguistique (Saussure)

Ferdinand de Saussure (1857–1913) définit le signe linguistique comme l'union de deux faces : ┌─────────────────────────┐ │ SIGNE │ │ ┌───────────────────┐ │ │ │ SIGNIFIANT │ │ ← image acoustique / forme sonore │ │ ex. /ʃɑ̃/ │ │ │ ├───────────────────┤ │ │ │ SIGNIFIÉ │ │ ← concept mental │ │ ex. [animal...] │ │ │ └───────────────────┘ │ │ ≠ │ │ RÉFÉRENT │ ← le chat réel dans le monde └─────────────────────────┘ Le lien signifiant/signifié est arbitraire (pas de ressemblance naturelle : le même animal s'appelle chat en fr., cat en angl., Katze en all.).

3. La double articulation (Martinet)

André Martinet montre que les langues humaines fonctionnent sur deux niveaux : 1ère articulation → unités AVEC sens (morphèmes/monèmes) ex. "cheval-ier-s" = 3 unités porteuses de sens 2ème articulation → unités SANS sens (phonèmes) qui distinguent les mots ex. /ʃ/ vs /b/ → "chat" ≠ "bas" Exemple concret "Les cavaliers" → 1ère articulation : le-s + caval + -ier + -s 2ème articulation : /l/, /e/, /k/, /a/, /v/, /a/, /l/, /j/, /e/ … chaque phonème peut changer le mot si on le remplace.

4. Les niveaux d'analyse linguistique

L'analyse linguistique se fait à 4 niveaux emboîtés : ① Phonologique → phonèmes /p/, /a/, /t/ ② Morphologique → morphèmes "chat" + "-s" (pluriel) ③ Syntaxique → syntagmes [GN] + [GV] ④ Clausal → clauses La pluie tombe.

5. La commutation : méthode clé

La commutation consiste à remplacer un élément par un autre pour tester s'ils appartiennent à la même unité. Si le remplacement change le sens = les deux éléments sont des unités distinctes. Exemple " p as" → " b as" : /p/ et /b/ sont deux phonèmes distincts (la commutation change le mot). "cha t " vs "cha p eau" : /t/ et /p/ sont deux phonèmes distincts. " -eur " dans "chanteur" et "danseur" → le même suffixe (commutation ne change pas la valeur : agent du procès). Paire minimale : deux mots qui ne diffèrent que par un seul phonème, prouvant que ce phonème est distinctif : son / ton / bon / mon / vont . Fiche de cours

La phonétique étudie les sons physiques (les phones). La phonologie étudie les sons distinctifs (les phonèmes) qui changent le sens dans une langue.

Une paire minimale prouve l'existence de deux phonèmes distincts : deux mots ne différant que d'un son (pain/bain). Le test s'appelle la commutation.

Deux phones qui sont des variantes du même phonème s'appellent des allophones (ex. : /r/ roulé et /r/ grasseyé = allophones de /ʁ/).

Le schwa /ə/ peut tomber sauf si sa chute crée 3 consonnes consécutives impossibles (règle des 3 consonnes).

La graphémique étudie l'écriture : alphagraphème (→ phonème : ‹ch›=/ʃ/), topogramme (ponctuation, majuscule), logogram (mot entier : & = « et »).

À ne pas confondre

Phonème ≠ lettre. ‹eau› = 3 lettres = 1 phonème /o/. ‹x› dans taxi = 1 lettre = 2 phonèmes /ks/.

Révision complète — D1 — Phonétique & phonologie
1. Phonétique vs phonologie

Phonétique : étude des sons réels produits et perçus (aspect physique). La transcription phonétique note tous les détails entre crochets : [pɛ̃] . Phonologie : étude des sons en tant qu'unités fonctionnelles du système de la langue. La transcription phonologique note les phonèmes entre barres obliques : /pɛ̃/ . Exemple clé Les /r/ de "Paris" et "rue" sont phonétiquement différents selon les locuteurs (roulé, uvulaire…) mais phonologiquement identiques : c'est le même phonème /ʁ/ en français standard.

2. Le système vocalique du français

Le français possède 16 voyelles (phonèmes vocaliques standard) : Antérieures non arrondies : /i/ vie · /e/ thé · /ɛ/ fête · /a/ patte · /ə/ le Antérieures arrondies : /y/ rue · /ø/ feu · /œ/ fleur Postérieures arrondies : /u/ doux · /o/ eau · /ɔ/ or Postérieure non arrondie : /ɑ/ pâte (tend à disparaître) Nasales : /ɛ̃/ pain · /ɑ̃/ pan · /ɔ̃/ pont · /œ̃/ un (tend à = /ɛ̃/)

3. Le système consonantique

Occlusives : /p/ pain · /b/ bain · /t/ ton · /d/ don · /k/ car · /g/ gar Fricatives : /f/ fou · /v/ vous · /s/ son · /z/ zon · /ʃ/ chat · /ʒ/ je Nasales : /m/ ma · /n/ nu · /ɲ/ gagne Liquides : /l/ loup · /ʁ/ rue Semi-cons. : /j/ yeux · /w/ oui · /ɥ/ nuit

4. Règles de transcription

On transcrit les sons , pas les lettres. "eau" → /o/ (3 lettres, 1 phonème). Le "e caduc" (/ə/) peut tomber à l'oral : "le" → [l(ə)] ou [l] selon le contexte. Les lettres muettes ne se transcrivent pas : "chat" → /ʃa/ (le ⟨t⟩ final est muet). La liaison est notée si elle se réalise : "les amis" → /lezami/. Exercice type "Pleure encor, pleure toutes tes ressources" (Rictus) Standard : /plFR plFR BkOR plFR tUt tC RCsURs/ (avec e caducs) Original : /plFR BkOR plFR tUt RsURs/ (sans e caducs = version apocope)

5. Traits distinctifs

Les phonèmes se définissent par des traits distinctifs (propriétés qui les différencient) : Voisement : /p/ (sourd) vs /b/ (sonore) — les cordes vocales vibrent pour /b/. Lieu d'articulation : /p/ (bilabial) vs /t/ (dental) vs /k/ (vélaire). Mode d'articulation : /p/ (occlusif) vs /f/ (fricatif). Astuce : /p/ – /b/ – /f/ – /v/ sont toutes des consonnes labiales. La différence : /p/ et /b/ sont occlusives (l'air est bloqué) ; /f/ et /v/ sont fricatives (l'air passe en frottant). Révision complète — D2 — Graphémique
1. Qu'est-ce qu'un graphème ?

Un graphème est la plus petite unité de l'écriture qui correspond à un phonème. Ce n'est pas toujours une seule lettre ! Type de graphème Exemple Phonème Monographe (1 l.) ⟨p⟩ → /p/ Digraphe (2 l.) ⟨ch⟩ → /ʃ/ | ⟨ph⟩ → /f/ Trigraphe (3 l.) ⟨eau⟩ → /o/ | ⟨ain⟩ → /ɛ̃/

2. Lettres muettes

Une lettre muette ne correspond à aucun phonème réalisé. Il en existe plusieurs catégories : ⟨h⟩ : toujours muet en français standard. "heure" → /œʁ/. Consonnes finales muettes : ⟨s⟩, ⟨t⟩, ⟨d⟩, ⟨p⟩, ⟨x⟩, ⟨z⟩ souvent muets en finale : "gris", "chat", "lit", "coup". ⟨e⟩ final : généralement muet (e caduc) : "table" → /tabl/. Lettres dans les digraphes : ⟨u⟩ de ⟨eau⟩, ⟨g⟩ de ⟨gn⟩ dans certaines positions. Exercice type "Châteaux" → combien de lettres muettes ? ⟨â⟩ → /ɑ/ (réalisé), ⟨t⟩ → /t/ (réalisé), ⟨e⟩ → muet, ⟨a⟩ dans ⟨eau⟩ → muet, ⟨u⟩ dans ⟨eau⟩ → muet, ⟨x⟩ final → muet (marque du pluriel) Transcription : /ʃɑto/ — 4 phonèmes pour 8 lettres

3. Correspondances graphème-phonème (CGP)

La relation entre graphèmes et phonèmes est complexe en français car : Un même phonème peut s'écrire de plusieurs façons : /o/ → ⟨o⟩, ⟨au⟩, ⟨eau⟩, ⟨ô⟩ Un même graphème peut correspondre à des phonèmes différents : ⟨c⟩ → /k/ (car) ou /s/ (ciel) Des groupes de lettres forment un seul phonème : ⟨ch⟩ → /ʃ/

4. La graphie du poème de Rictus

Jehan Rictus (1867–1933) utilise une graphie non-standard pour représenter l'oralité populaire parisienne. Il remplace les e caducs facultatifs par des apostrophes : Pleur' ! Pleure encor, pleur' tout's tes r'ssources Principe : l'apostrophe indique la chute du /ə/ à l'oral. Le /ə/ est maintenu (écrit) dans 3 cas : En fin de vers (accent possible : "Pleure" avec accent) Avant une voyelle (liaison impossible sans lui) Quand sa chute crée un groupe consonantique impossible (≥ 3 consonnes) : "le glas" (*Rlgl) Effet stylistique : la graphie apostrophique impose une diction populaire et signale que le locuteur appartient aux classes populaires parisiennes du début du XXe s.

5. Graphèmes contextuels

Certains graphèmes changent de valeur phonique selon leur contexte : ⟨c⟩ → /k/ devant ⟨a, o, u⟩ (car, coq, cul) ; → /s/ devant ⟨e, i, y⟩ (ciel, ceci) ⟨g⟩ → /ɡ/ devant ⟨a, o, u⟩ (gare) ; → /ʒ/ devant ⟨e, i, y⟩ (genre, gilet) ⟨s⟩ → /s/ en début de mot ou après consonne ; → /z/ entre deux voyelles (maison /mezɔ̃/) Types de graphèmes Type Graphème Phonème Exemple Monographe ⟨p⟩ /p/ pain Digraphe ⟨ch⟩ /ʃ/ chat Digraphe ⟨ph⟩ /f/ photo Digraphe ⟨gn⟩ /ɲ/ gagne Digraphe ⟨ou⟩ /u/ doux Digraphe ⟨au⟩ /o/ faux Trigraphe ⟨eau⟩ /o/ beau Trigraphe ⟨ain⟩ /ɛ̃/ pain Graphèmes contextuels : ⟨c⟩ et ⟨g⟩ Graphème Contexte Phonème Exemple ⟨c⟩ + a, o, u /k/ car, coq, cul ⟨c⟩ + e, i, y /s/ ciel, ceci, cycle ⟨ç⟩ + a, …

La morphologie étudie la structure interne des mots. L'unité de base est le morphème : la plus petite unité porteuse de sens. Dans chanteurs : {chant-} + {-eur} + {-s} = 3 morphèmes.

Révision complète — D7 — Extensions du groupe nominal
1. Le groupe nominal étendu

Le GN canonique = Déterminant + Nom. Les extensions du GN sont des constituants qui s'ajoutent au nom-tête pour le préciser, le qualifier ou l'identifier. GN étendu = [Dét] + [Épith. antéposée] + N + [Épith. postposée] + [Complément du nom] + [Proposition relative]

2. Épithète intégrée

Adjectif directement rattaché au nom-tête, sans virgule ni pause : Antéposée vs Postposée Postposée (position canonique) : valeur classifiante , objective, descriptive. → "une voiture rouge " = la classe des voitures rouges (≠ voitures bleues) Antéposée : valeur subjective , affective, expressive. → "une belle journée" = appréciation personnelle du locuteur Opposition de sens possible : "un grand homme" (remarquable) ≠ "un homme grand " (de haute taille)

3. Épithète détachée

Adjectif séparé du GN par une virgule (et une pause à l'oral). Valeur prédicative secondaire — elle prédie une propriété en plus de la prédication principale. Épuisé , il s'endormit aussitôt. (= parce qu'il était épuisé) Elle marchait, les yeux grand ouverts , sans rien voir. Valeurs possibles : cause, temps, concession, manière — toujours inférées du contexte. L'épithète détachée est très mobile : elle peut se placer en tête, en milieu ou en fin de clause.

4. Apposition

GN (ou équivalent) accolé à un autre GN, séparé par une virgule, qui identifie ou reformule le référent : Sartre, philosophe existentialiste , écrit sur la liberté. Paris, la Ville Lumière , accueille des millions de touristes. À distinguer : épithète détachée = adjectif ; apposition = GN. Les deux ont une virgule et une valeur prédicative secondaire, mais la catégorie grammaticale diffère.

5. Construction absolue

Syntagme détaché (GN + participe ou adjectif) sans verbe conjugué, à valeur circonstancielle : La nuit tombée , ils rentrèrent. (= quand la nuit fut tombée → temps) Les bras croisés , il attendait. (→ manière) Son travail terminé , elle rentra. (→ cause) La construction absolue est à distinguer de l'épithète détachée : CA = GN + participe (mini-syntagme) ; épithète = adjectif seul.

6. Complément du nom (CN)

GN introduit par une préposition (le plus souvent de ) qui caractérise le nom-tête : la maison de mes parents (possession) | un vase de cristal (matière) | une idée de génie (qualité)

Révision complète — D10 — Phrase graphique & dislocation
1. La phrase graphique

La phrase graphique est une unité de l'écriture délimitée par une majuscule initiale et un point final (., !, ?, …). Elle n'est pas une unité syntaxique : elle peut contenir plusieurs clauses ou être une simple clause averbale. Exemples "Magnifique !" = 1 phrase graphique, 1 clause averbale. "Il pleut et je reste chez moi, mais demain il fera beau." = 1 phrase graphique, 3 clauses. "Dès qu'il arriva, Marie, qui l'attendait depuis des heures, se leva." = 1 phrase graphique, 3 clauses (principale + 2 subordonnées).

2. La dislocation

La dislocation déplace un constituant en périphérie (gauche ou droite) de la clause, avec reprise par un pronom à l'intérieur : Dislocation gauche : X, [pronom] V… Dislocation droite : … [pronom] V, X Dislocation gauche (DG) Paul , je le connais bien. → "Paul" est disloqué à gauche, repris par "le" (cataphorique). Valeur DI : thématisation (Paul = thème, info donnée). Dislocation droite (DD) Je le connais bien, Paul . → "Paul" est disloqué à droite, repris par "le" (anaphorique). Valeur DI : rhématisation / précision après coup. Exemples à l'oral DG : "Ce film, je l'ai déjà vu." / "Ta sœur, elle est vraiment sympa." DD : "Il est sympa, ton frère." / "Je l'aime bien, cette chanson."

3. Clivage

La clause clivée ( c'est X qui/que ) focalise un constituant en position rhématique forte : C'est Paul qui a dit ça. (Paul = focus rhématique contrastif) Différence avec la DG : DG thématise (info donnée) ; clivage focalise (rhème fort, souvent contrastif).

4. Inversion du sujet

Inversion simple : pronom sujet postposé → "Viendra-t-il ?" Inversion complexe : SN maintenu + pronom de reprise → "Paul viendra-t-il ?" Inversion stylistique : SN postposé, registre soutenu/littéraire → "Ainsi parla l'orateur." Valeur DI : le sujet postposé occupe une position rhématique (information nouvelle). Registre : la dislocation est très fréquente à l'oral et dans le registre familier. En littérature, elle peut mimer l'oral des personnages (style indirect libre chez Zola) ou créer un effet de proximité avec le lecteur.

Exercice 2 — Oral vs écrit — marques du pluriel
Transcription : /le zɑ̃fɑ̃ mɑ̃ʒɛ de pɛ̃ o ʃɔkɔla/

À l'écrit : pluriel marqué sur « les », « enfants » (-s), « mangeaient » (-aient), « des », « pains » (-s).
À l'oral : seul le déterminant « les » (/le/) vs « le » (/lə/) marque le pluriel. La liaison les-z-enfants (/le zɑ̃fɑ̃/) aussi. Les -s finaux de « enfants » et « pains » ne se prononcent pas. La désinence « -aient » est homophones de « -ait » → le pluriel verbal est neutralisé à l'oral.

Conclusion : le français oral est beaucoup moins redondant que l'écrit pour les marques du pluriel.

technologie : techno- (grec : art, technique) + -logie (science de) → science de la technique
dépollution : dé- (préfixe = inverse) + pollution → action d'éliminer la pollution

Champs sémantiques : ① santé / médecine (bronchiolite, hyperréactivité) ② environnement (dépollution, NOx, émissions) ③ technologie / innovation (technologie, système, AdBlue) ④ performance (réduction de 90%, normes européennes)

Valence verbale et rôles actantiels

Corrigés
Exercice 1 — Analyse de valence — constructions régulières et irrégulières
1. hennir : verbe intransitif (0 actant ou 1 actant : le cheval) → chez Apollinaire, il prend un COD (« tout un univers »). Effet : transitivation poétique — les nuages deviennent des entités qui « émettent » un espace urbain, créant une métaphore synesthésique.

2. émaner : standard = intransitif + de (source) → chez Céline, il devient transitif direct (« émanais des odeurs »). Effet : registre familier / langue parlée, transgression syntaxique volontaire.

3. parodier : transitif standard → emploi standard chez Aragon. inventer : transactif (créer quelque chose) → chez Balzac, emploi absolu ou au sens « découvrir ». Effet : glissement sémantique (inventer qch ≠ découvrir qch).
Exercice 2 — Valence de trembler — le pronom « vous »
La valence standard de trembler est monovalente (1 actant : le sujet qui tremble).

Ici, la construction est : [sujet : l'allumette] + [vous] + tremble + [entre les doigts]
« vous » est un datif éthique (ou datif d'intérêt) : pronom personnel qui n'est pas un actant valenciel mais marque l'implication affective du locuteur/narrateur dans l'action.

Fonction syntaxique : complément indirect (lui, vous = « dans votre espace, à votre détriment »)
Rôle actantiel : affecté / bénéficiaire indirect

Effet dans le récit : le pronom « vous » interpelle directement le lecteur, le plonge dans l'action, crée une narration à la 2e personne implicite. L'expérience devient universelle et immédiate. Procédé d'implication du lecteur (technique de la focalisation 0 → 2).
Exercice 3 — Rôles actantiels dans Michaux — « Le Grand Combat »
Particularité : tous les verbes sont des mots inventés (néologismes : emparouiller, endosquer, rague, roupète, pratèle, libucquer, baruffle, tocarde, marmine…).

Compréhension malgré tout : grâce aux rôles actantiels :
• Agent constant : « il » (sujet) — l'agresseur
• Patient constant : « le / l' » (COD) — la victime
• Les verbes transitifs impliquent une action d'un sur l'autre
• Syntagmes prépositionnels (« contre terre », « jusqu'à son drâle ») donnent des indications spatiales
• Structure narrative : escalade de violence → « Enfin il l'écorcobalisse » (issue finale)

Intelligibilité : la structure syntaxique (SVO) + les rôles actantiels (agent/patient) + les connecteurs (« Enfin ») maintiennent la compréhension globale de la scène de combat.
